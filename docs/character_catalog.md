> ⚠️ **重要**：此素材包**所有动作都是正面单方向**（chibi 风格固有限制）。  
> 没有"朝上 / 朝下 / 朝侧"的方向行——所有方向用同一个正面动画。

### 朝向逻辑（左右镜像）

由于素材本身只有正面动画，朝向通过 `flip_h` 实现：

```gdscript
# 朝右 / 朝下 / 朝上：flip_h = false
# 朝左：           flip_h = true
# idle 时保持上次的 flip_h（角色不会突然转回正面）
```

`SpriteFactory.direction_needs_flip(dir)` 工具函数返回是否需翻转。

## 五、Godot 接入建议

每个角色资源在 Godot 里这样用：

1. 把 PNG 拖进 `assets/characters/` 目录
2. 角色场景里用 `AnimatedSprite2D` 节点
3. 在 `_ready()` 里调 `SpriteFactory.build_frames_from_path(sprite_file)` 拿到 `SpriteFrames`，赋给 `AnimatedSprite2D.sprite_frames`
4. 在 `_physics_process` 里根据 `velocity` 切动画：

```gdscript
var dir := SpriteFactory.direction_from_velocity(velocity)
if dir == "":
    sprite.play("idle")
else:
    if dir == "left" or dir == "right":
        sprite.flip_h = (dir == "left")
    sprite.play("walk")
```
